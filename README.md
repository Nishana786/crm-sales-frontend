# CRM Sales Frontend 🚀

A React Native (Expo) based CRM application for managing sales leads.

## 📌 Current Features
- Add new leads
- View leads list
- Lead detail page
- Basic analytics & profile screens (UI)

## 🚧 Work in Progress
- Edit & delete leads
- Backend integration
- Authentication improvements
- Deployment (APK build)

## 🛠 Tech Stack
- React Native (Expo)
- TypeScript
- Context API
- Tailwind CSS

## ▶️ Run the App
```bash
npm install
npx expo start
